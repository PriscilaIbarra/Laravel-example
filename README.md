# Test
CRUD example of users and roles with laravel
###  Instalation:

 ```npm install```
 
 ```npm run dev```
 
 ```npm run watch```
 
 ```composer install```
 
 ```composer dumpautoload```
 
 ```Create .env (configuration file)```
 
 ```php artisan key:generate --show```
 
 ```php artisan migrate:fresh```
 
 ```php artisan db:seed```
